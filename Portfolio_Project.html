Noise Canselling


Making Film

Project Planning, Hardware Developer & Engineer

Interactive media installation | TeamWork.AUJIL

DESCRIPTION

사람들은 바깥에서 들려오는 소리를 듣고 싶지 않을 때 '노이즈 캔슬링'이라는 기능을 사용한다. 기술의 이면과 그 위험성을 알면서도 그저 외면하고 살아가는 현상을 ‘온 세상에 걸린 노이즈 캔슬링’이라고 바라보았다.

작품은 천장에 달린 5M * 3M 크기의 스크린, 바닥 중앙에 배치된 스피커 그리고 4개의 헤드폰으로 구성되어 있다. 스크린은 관객을 향해 약간(15도) 기울어 있는 상태로 매달려 있다. 관객의 시선과 같은 방향에서 빔 프로젝터를 맵핑하여 스크린에 콘텐츠를 영사한다. 콘텐츠의 내용은 헤드폰을 착용하고 있는 관객 수에 따라 달라지며, 발생하는 변화는 다음과 같다. 헤드폰을 아무도 쓰고 있지 않을 때에는, 날씨가 흐리고 비가 오며(눈이 오며) 하늘에서 많은 물건이 떨어진다. 시야가 흐려 하늘에 거대한 무언가가 희미하게 보인다. 헤드폰을 쓰는 사람이 늘어날 때마다 (비행기 띵 소리 들려주고) 이 사운드와 함께 떨어지는 물건의 수가 줄어들며, 날씨가 점점 맑아진다. 상공에 떠있는 기체가 점점 선명히 보인다. 헤드폰을 착용하고 있는 사람이 줄어들면 화면 속 기체가 흔들리며 날씨가 흐려지고, 떨어지는 물체의 양이 다시 늘어나게 된다. 4개의 헤드폰이 모두 착용 중인 상태로 일정 시간이 지나면 물건이 더 이상 떨어지지 않게 된다. 또한 유리의 금이 사라지며 상태가 초기화 된다. 바닥 중앙의 스피커에서는 배경음악과 함께 물체가 유리에 부딪히는 소리가 먹먹하게 된다. 일렬로 배치된 4개의 헤드폰에서는 물체가 부딪히는 소리를 비롯해 심장박동 소리, 거친 호흡소리. 이명 등 오래 듣기 힘들고 거북한 사운드가 들려온다.

Technical Director, Hardware Developer, Engineer

Interactive media installation | TeamWork

DESCRIPTION

작품은 헤드폰과 헤드폰을 착용 함에 따른 LED Strip, Stepper 모터의 인터렉션 요소로 구성된다. LED Strip에는 헤드폰 기울기 (Roll) 에 따른 빛의 움직임 변화가 표현되고, 특정 센서 값에서 스텝 모터가 구동하는 구성이다.

좌대 위에 본 작품이 있으며, 관객은 헤드폰 거치대에 거치되어 있는 헤드폰을 착용하여 작품과 상호 작용 할 수 있다. 1. 헤드폰 착용 시 작품 속 Led Strip 가운데에 불빛이 점등한다.
2. 헤드폰 속에서는 미리 제작 된 사운드가 재생된다.
3. 머리를 좌우 (Roll)로 기울이면 센서 값에 따라 led 불빛이 실시간으로 움직인다.
4. led 불빛이 좌우 끝에 다다를 경우, 그에 해당하는 모터를 구동 하며 천천히 돌아간다.

[개발과정]
# 송신부 개발보드
완성도를 위해 헤드폰 안에 배터리와 가속도-자이로 센서, 리밋 스위치를 넣고 봉합하였다. 리밋 스위치를 통해 착용 여부를 감지하고, 6축 IMU 센서를 통해 헤드폰의 Roll 값을 센싱한다. 센싱 받은 값은 RF (Radio Frequence) 433mhz 주파수 대역의 통신을 통해 데이터 값을 송신한다.

# 수신부 개발보드
RF통신 모듈의 수신부로 받아 버퍼로 들어온 데이터 값은 string 타입으로 받아온다. 이후, string 타입의 문자를 int 형으로 센싱 되는 범위를 정해주어야 하기 때문에, int 형 타입으로 형변환 시켜준다. Led 라이브러리는 FastLED 라이브러리를 이용했다. 총 30개의 led 개수를 사용했고, int형 타입의 데이터 값을 송신부 개발보드에서 값을 알맞게 처리해주는 과정을 거쳤기 때문에, 수신부 보드에서는 계산 처리 과정 없이 파싱했다.

1스텝 당 1.8도의 각도를 지원하는 12V 스텝 모터를 사용했다. 모터 드라이브는 1/16 분해능까지 가능한 A4988 모듈을 사용하고, 2개의 스텝 모터를 사용하기에 모터 드라이브도 2개를 사용했다. Led strip 의 동작하는 값이 좌측에 다다르면 좌측 모터가, 우측에 다다르면 우측 모터가 구동하도록 소스코드를 구성했다.

Realtime Data Visualization


Director, Producer

DataVisual

DESCRIPTION

"로우 데이터 가공과 처리, 그래픽 작업, 사운드 작업을 한 데이터 시각화 입니다. 데이터는 케플러 우주 망원경에서 파생된 데이터를 사용하였고, 그 중에서도 시간에 따라 변화하는 570여개 항성들의 광도 값을 나타낸 로우 데이터를 사용했습니다. 수치로만 나열되어 있는 데이터의 심미성을 연구하고 본인만의 작업으로 표현을 해보고자, 노드 방식을 통해 실시간으로 그래픽을 출력 할 수 있는 비주얼 프로그래밍 언어 ‘TouchDesigner’ 를 활용했습니다. TouchDesigner 에서는 데이터를 parsing 하고 작업환경에 적합하게 만들기 위해 가공하는 과정과 가공된 데이터를 시각화하는 작업을 가졌습니다.

세부적으로 보면, 데이터를 프로그램이 반응 할 수 있는 최소 범위, 최대 범위로 수치를 ( 입력 받는 데이터 ) / 100 으로 치환했습니다. 그리고 치환된 데이터 값을 실시간 그래픽으로 출력하기 위해 일정한 간격으로 현재 데이터 값에서 다음 데이터 값을 순차적으로 처리하는 구조( lfo, math 노드사용 ) 를 제작했습니다. 작업에서 기본이 되는 그래픽 요소는 Noise( 그래픽 입자 ) 와 Feedback( 잔상 ) 입니다, 앞에서 먼저 만들어 놓은 그래픽 입자들을 Ramp 노드로 그래픽 패턴을 만들어 데이터 입력 값에 실시간으로 반응하는 파동 형태의 패턴을 제작하였습니다. 이 후, projection, transform 노드들을 통해 3가지 형태의 파동 모델을 만들었습니다. 그리고 터치디자이너에서 출력한 3가지 형태로 만든 파동 모델을 활용해, 마치 항성의 외부에서 내부 핵까지들어갔다 나오며 미지의 세계를 탐험 하듯 연출 기법을 표현하고자, Adobe사의 After Effect 프로그램을 사용하였습니다.

마지막으로 작,편곡 작업을 해왔기 때문에, 이번 작품에서도 시각화에 맞는 사운드를 제작했습니다. 사운드 제작은 Logic 프로그램을 사용하였고, 주로 사용된 가상악기에는 우주 과학의 경이로운 느낌을 자아내기 위해 다양한 FX 사운드가 있는 옴니스파이어와 피아노 가상악기인 키스케이프, 트릴리안 베이스를 사용했습니다. 이 외에는 여러 샘플링 작업을 통해 제작했습니다. 그리고 작품의 창작 의도와 최종적으로 제작된 데이터 시각화 영상 의도가 잘 어우러지게끔 하기 위해 작업의 가장 마지막 포스트 프로덕션 과정에서 완성됐습니다. 특히, 작품은 항성의 표면에서 내부 핵까지 탐험하는 듯한 연출을 나타내려 했기에 이러한 요소들의 효과를 극대화하기 위해 사운드 제작에도 신경을 많이 썼습니다."

Interactive Installation | TeamWork, Art Prodution

DESCRIPTION

물체는 서로 끌어당기고 밀쳐내는 힘에 의해 그 형태를 유지한다. 그러한 관점에서 어떤 사물이란 그 사물 그 자체로써 존재하는 것이 아닌, 시간선 위에서 힘의 흐름이 어떤 형태를 비교적 오래 유지하고 있는 것이다. 작품은 관객에게 무한히 회전하는 액체 금속의 모습을 보여준다. 작품을 처음 마주했을 때는 마치 액체 금속이 움직이는 것처럼 보이지만, 실제로는 액체 금속이 아닌 액체 금속을 들어 올리는 자기력이 이동하는 것일 뿐이다. 이를 통해 작품은 사물의 형태를 유지하고 변화시키는 힘의 움직임을 시각적으로 함축해 묘사한다. 여기서 관객은 소리를 통해 액체 금속의 회전 속도를 변화시킬 수 있는데, 이를 통해 자기력의 움직임이 더욱 분명하게 드러나고 관객은 작품의 반복적이면서 불규칙한 동세를 관찰하며 명상적 체험을 하게 된다.

하얀 테이블 위 뾰족뾰족한 검은 물체가 원을 그리며 회전한다. 검은 물체는 지나온 자리에 검은 길을 남기고 조명은 물체와 물체가 남긴 길을 비춘다. 잔잔하고 진중한 배경음악이 들려오고 관객은 작품 근처를 무심하게 지나며 작품을 감상한다. 관객이 대화를 나누거나 소리를 내면 검은 물체는 관객의 소리에 반응해 회전하는 속도를 달리 한다. 관객은 작품과 상호작용하며 물체의 움직임을 관찰한다.
